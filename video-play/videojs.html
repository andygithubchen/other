Nuevo plugin and theme for Video Js player

Nuevolab presents custom theme and Nuevo plugin for Video Js player. Sponsored by Brightcove Video.js player framework is the project built by the community, which can handle the most professional use cases and is used on thousands of video websites, including such brands as Instagram, Twitter, Windows, Dropbox, TheGuardian, Puma, Hugo Boss, Symantec, AccuWather, Sure and many more.
Open source library for working with video on the web with dozens available plugins makes Video.js most advanced and most popular HTML5 player today.

What is Nuevo theme/plugin ?

Nuevo theme for video.js is completely custom design optimized for mobile devices based on custom css stylesheet and javascript code. The default color schema of Nuevo theme is light blue. You can easily change it by setting appropriate option in player's setup code.

Nuevo plugin is set of advanced player features merged into one javascript file plugin, 45 kB only file size. It contains options for: watermark image, playback speed control, zoom effect, sharing options, related videos and video resolution switcher - ALL IN ONE !

For Nuevo theme and plugin options, player setup code examples please check dedicated online documentation.
